The *Consumer Chemicals and Containers Regulations, 2001* (R.S., c. 24 (3rd Supp.), s. 1) are a set of detailed safety regulations designed to protect consumers from the risks associated with hazardous chemical products and their containers. These regulations establish standardized requirements for:
- Hazard classification,
- Labeling and information display,
- Container design and safety features (e.g., child resistance),
- Testing procedures,
- Data sources for hazard evaluation,
- Compliance and recordkeeping.

> **Important Note**: These regulations **do not apply to vaping products**, as defined in the *Tobacco and Vaping Products Act*.

## ðŸš¨ Hazard Classification & Evaluation

### 1. **Hazard Determination Process**
The responsible person must:
- Determine the **hazard categories** and **sub-categories** of the chemical product or container.
- Identify the **required container type**.
- Determine the **required labeling and information** to be displayed.

> ðŸ” **Multiple Hazards**: If a product falls into more than one hazard category, **all required information must be displayed** on the container.

> âš ï¸ **Sub-category Priority**: If a product falls into multiple sub-categories within one category (e.g., "spontaneously combustible" and "flammable"), it **must be classified in all** â€” not just the most severe.

> ðŸš¨ **Multiple Exposure Routes**: If exposure can occur via multiple routes (e.g., ingestion, inhalation), **information must be provided for each route**.

### 2. **Data Sources for Hazard Assessment (in order of precedence)**

| Source | Description |
|-------|-------------|
| 1. **Human Experience Data** | Peer-reviewed data showing actual injuries or poisonings (especially child exposure). |
| 2. **OECD Test Guidelines** | Peer-reviewed, standardized animal or in vitro testing (e.g., LD50, LC50). |
| 3. **Tests under these Regulations** | If OECD tests were not done, then tests conducted per these regulations. |
| 4. **National or International Standards** | Recognized standards (e.g., ISO, CSA) that meet good scientific practices. |
| 5. **Generally Accepted Procedures** | Methods following good scientific practices at time of testing. |
| 6. **Similar Products** | Data on products with similar properties (e.g., structure, use, ingredients). |
| 7. **Other Peer-Reviewed Information** | Current scientific literature on the product or its ingredients. |
| 8. **Product-Specific Testing** | Tests conducted by the responsible person using good scientific practices (if no other data available). |

> âœ… **Preference Rules**:
> - Original literature > secondary references.
> - Data showing **greatest hazard** takes precedence.
